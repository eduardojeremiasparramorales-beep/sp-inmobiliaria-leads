⬇ Descargar e instalar
  1. Toca Descargar e instalar (si estás en WhatsApp, primero abre esta página en Chrome: menú ⋮ → "Abrir en Chrome").
  2. Cuando termine la descarga, baja la barra de notificaciones y toca el archivo (o Chrome → ⋮ → Descargas).
  3. Si el teléfono pregunta, activa "Permitir instalar aplicaciones desconocidas" para Chrome y vuelve atrás.
  4. Toca Instalar y abre la app. Entra con tu teléfono y PIN.
¿Dice "App no instalada"? Tienes una versión vieja con otra firma: desinstala Leons Group primero y repite. Esto pasa solo esta vez — desde esta versión la app se actualiza sola, sin desinstalar nada.
